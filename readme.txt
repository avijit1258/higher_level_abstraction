Please, follow these steps to run.

In your command prompt/terminal, go to your preferred directory and type 'git clone https://github.com/avijit1258/higher_level_abstraction.git'
Then, go inside the folder 'higher_level_abstraction' and type 'cd higher_level_abstraction'
Create a virtual environment using this command 'python3 -m venv virtual_env' in your terminal/command prompt
Once virtual environment has been created, run the virtual environment using this command 'source virtual_env/bin/activate'
Install required packages/libraries using this command 'pip install -r requirements.txt'
Please, make sure that every requirement packages have been installed properly
Now, just run 'tool/app.py' to run the project.
